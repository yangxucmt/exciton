# Figure S3 Data and Plotting Code

This folder contains the code and data used to generate Fig. S3.

## Overview

- Fig. S3a–c: Free particle–hole shift vector at three equivalent $K$ points.

---

## Fig. S3a — $\theta=0$ (zero strain)

Data file: "figs3_theta_0.csv"

- "kx_list": $k_xa$ at three equivalent $K$ points ($a=3.16\text{\AA}$).
- "ky_list": $k_ya$ at three equivalent $K$ points.
- "rx_list": $x$ component of the shift vector at $\theta = 0$, scaled by $400$ (unit: $a/400$).
- "ry_list": $y$ component of the shift vector at $\theta = 0$, scaled by $400$ (unit: $a/400$).

Computed at zero strain for transitions from the valence band to the lower conduction band for spin-up electrons.
